# DinoDash

DinoDash es un juego programado con la librería de PyGames cuyo objetivo es atravesar el máximo de plataformas posibles mediante la creación de puentes de crecimiento dinámico. 

📄 **[Leer el Manual de Usuario (PDF)](./docs/Manual_Usuario_DinoDash.pdf)**

## 🛠️ Tecnologías utilizadas

* **Lenguaje:** Python
* **Librerías/Frameworks:** PyGames
